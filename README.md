**This project is developed using C++ and includes a menu-driven system that allows the user to work with different data structures.
**

->Data Structures Used
  Linked List
  Stack (Dynamic Array)
  Priority Queue
->Features
 Add / Insert data
 Delete data
 Display / Print data
 Search and manage data
 Reverse linked list
 Count nodes
 Check if stack is empty
 Enqueue and dequeue in priority queue
 Exit menu system
->File Handling
 The program reads data from files at startup.
 Data is saved back to files before exiting.
